# 目标读者画像：AI 应用架构师与基础设施工程师

## 🛠️ 核心关注点（极高权重）
1. 开源生态：具备 GitHub 仓库、Hugging Face 权重、模型架构完全开源、或可本地化部署的项目。
2. 硬核技术创新：底层架构演进（如新型 MoE 拓扑、注意力机制改进、创新的合成数据清洗策略、创新的量化/微调技术）。
3. 性能基准：具体的测试集表现（如 MMLU-Pro、HumanEval、Math-2024 等跑分），以及推理延迟（Latency）、吞吐量（Throughput）和显存占用。
4. 落地实用性：新开放的 API Endpoint、SDK 更新、降价公告、Function Calling 响应准确率提升、长文本上下文（Context Window）的实际利用效率。

## ❌ 绝对排斥点（零权重/负分降级）
1. 公关八卦与软文：各大厂没有技术白皮书支撑的“战略签约”、“未来愿景”、“首发大模型预告”。
2. 营销话术：全篇充满“颠覆性”、“里程碑”、“史诗级行业变革”等缺乏客观数据支撑的吹嘘。
3. 纯资本游戏：与降低开发者 API 成本、底层基础设施演进完全无关的流言和无实质业务变动的融资细节。
